# 🤖 AI Atlas

**AI Tools Catalog for Developers**

A beautiful, modern single-page application showcasing the best AI tools for developers. Built with React and designed with a stunning dark theme.

## ✨ Features

- 🔍 **Smart Search** - Find tools by name, description, or tags
- 🏷️ **Category Filtering** - Filter by tool categories
- 📱 **Responsive Design** - Works perfectly on all devices
- 🎨 **Beautiful UI** - Modern dark theme with gradients and animations
- ⚡ **Fast Performance** - Optimized React components
- 🚀 **GitHub Pages Ready** - Automatic deployment with GitHub Actions

## 🛠️ Tools Featured

Our catalog includes 10 carefully selected AI tools:

1. **GitHub Copilot** - AI pair programmer
2. **ChatGPT** - Conversational AI for code help
3. **Tabnine** - AI code completion
4. **Codeium** - Free AI code acceleration
5. **Replit Ghostwriter** - AI pair programmer for Replit
6. **Amazon CodeWhisperer** - ML-powered code generator
7. **DeepCode** - AI-powered code review
8. **Mintlify** - AI documentation writer
9. **Cursor** - AI-first code editor
10. **Sourcegraph Cody** - AI coding assistant

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/khabibulllin/ai-atlas.git

# Navigate to project directory
cd ai-atlas

# Install dependencies
npm install

# Start development server
npm start
```

The app will be available at `http://localhost:3000`

### Building for Production

```bash
# Create production build
npm run build

# The build folder contains the production-ready files
```

## 🌐 Deployment

This project is configured for automatic deployment to GitHub Pages using GitHub Actions. Every push to the main branch will trigger a new deployment.

### Manual Deployment

```bash
# Build and deploy to GitHub Pages
npm run deploy
```
